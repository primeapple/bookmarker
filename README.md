# bookmarker
It's [harpoon](https://github.com/ThePrimeagen/harpoon) for your shell.

## Installation

1. Install the `bm` binary

### Releases

```sh
go install github.com/primeapple/bookmarker/cmd/bm@latest
```

### Locally

```sh
make install
```

## Setup

Connect your shell
    1. Fish
    ```sh
    # add to your config.fish
    bm init fish | source
    ```
